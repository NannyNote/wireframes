🔄 Consistent pattern

The ✎ time pill is the single entry point for both real-time and retrospective logging — no separate link needed. The nanny learns this interaction once on the meal sheet and recognises it everywhere.

💡 Info strip

A small sky blue hint below the timer reads "Already put down? Tap the time above to set when the nap started." Surfaced only on the sleep sheet — the only log where a timer is involved. Guides without cluttering.

⏱ How the timer works retrospectively

Nanny taps ✎ pill → sets start time to 13:10 → taps Done → timer counts forward from 13:10 → when Olivia wakes the nanny taps Stop → duration calculated from corrected start to actual stop.

💾 Saved identically

Whether logged live or retrospectively, the saved data is the same: { start: "13:10", end: "15:01", duration_mins: 111 }. No flag. No difference visible in the diary or parent view.
