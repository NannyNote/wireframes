Baby view — tummy time

  • Tummy time card auto-shown for babies under 12 months
  • Duration chips: None · 5–10m · 10–20m · 20+m
  • Description is optional for activity — just type matters for filtering
  • Save active once at least one type selected
🎨 Multi-select chips

Multiple activity types can be selected — each taps to toggle on/off with a sunshine yellow fill. At least one must be selected before saving. The selected types determine the colour coding and filtering in the recent log.

📋 Plan pull-through

Same pattern as the meal sheet — planned activity pulled from weekly_plans, "We did something different" checkbox below. Same strikethrough + fade when overridden. Only what happened saved — deviation stays private.

📝 Description is optional

Unlike the meal sheet, description is optional for activities. The type chips alone are enough to log — the nanny only adds a description when there's something worth capturing. Placeholder reads "e.g. made a bird feeder from a toilet roll".

🍼 Tummy time

Tummy time card auto-shown for babies under 12 months based on child profile age. Hidden for toddlers and older children. Duration chips only — no timer, no notes. Clinically important so it gets its own card rather than being buried in the type chips.

💾 What gets saved

{ category: "activity", types: ["outdoor","creative"], description: "Nature walk + leaf printing", duration: "1hr_plus", planned: true, used_plan: true } or with override: types: ["creative","sensory"], description: "Playdough animals", used_plan: false }
