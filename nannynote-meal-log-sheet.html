✎ Editable time

The time pill in the sheet header has a pencil icon — clearly tappable. Tapping turns it coral and drops in a simple hour/minute picker below the header. A hint note reassures the nanny that the original logged time is preserved. Tap Done to update.

📷 Photo placement

The photo section is a designed part of the sheet — it sits between the log fields and the Save button, separated by a divider. Camera icon + caption input on one row. Not an add-on, not an afterthought.

🖼️ Multiple photos

Up to 3 photos per entry. Thumbnails sit in a horizontal row with a small ➕ camera after them while slots remain. One shared caption for all photos. ✕ on each thumbnail to remove individually.

💬 Caption is optional

The caption input placeholder reads "Add a caption..." — entirely optional. Many nannies will save without a caption. When filled, the caption appears in the parent view alongside the photos.

💾 What gets saved

{ category: "meal", type: "lunch", eaten: "Scrambled eggs on toast", amount: "ate_well", time: "11:45", photos: [{url: "...", caption: "Proud of her clean plate!"}] }
