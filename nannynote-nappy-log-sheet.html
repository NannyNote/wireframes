⚡ 3 taps max

The fastest log in the app. Open sheet → tap contents → tap Save. That's it. Nappy rash is optional — most nappy changes won't need it. The rash prompt is always visible but stays compact unless Yes is tapped.

🎨 Contents chip colours

Each chip has its own subtle accent: Wet → sky blue · Dirty → gold · Both → mint · Dry → forest green. These aren't loud — just enough visual differentiation to scan quickly in the diary view.

🔴 Rash section

Tapping Yes expands the section to show None · Mild · Moderate chips. Mild is pre-selected as the most likely. The rash section has a warm red tint — visually distinct in the diary so it's easy to spot a rash developing over multiple entries.

🚫 No photo

The only log type without a photo attachment. Nappy changes aren't shareable moments — removing the camera icon keeps the sheet focused and the interaction fast.

🟢 Mint save button

The Save button uses mint green rather than forest green — matches the sheet's category colour for visual consistency. The same principle will apply to all log sheets: the save button reflects the category accent.

💾 Saved data

{ category: "nappy", contents: "wet", rash: false } or with rash: { category: "nappy", contents: "dirty", rash: true, rash_severity: "mild" }
