📋 Bottom sheet

Slides up over the home screen exactly like the meal and nappy log. Drag handle at top, home screen dimmed behind. Consistent with all other quick log sheets.

🔀 Conditional logic

Breast → Side + Duration. Bottle → Amount + Milk type. Fields appear and disappear completely — no greyed-out states, no hidden rows.

🤱 Side pre-selection

Always pre-selects the opposite side to the last recorded breast feed. Last was Left → Right pre-selected. Last was Right → Left pre-selected. Last was Both → Both pre-selected. Nanny can override.

⏱️ Duration

Manual stepper only for Phase 1 — nanny logs retrospectively after the feed. No live timer in Phase 1. Live breast feed timer (background-persistent, like sleep) is a Phase 2 feature.

💾 Data

Breast: { type:"breast", side:"right", duration_mins:12 }. Bottle: { type:"bottle", amount_ml:120, unit:"ml", milk_type:"breast_milk" }.
